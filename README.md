# Cat Speed

This game uses the arrow keys and ASWD to move the little kitty.
Try to collect the 2 coins.

Jump of of the platform to proceed to the next level. Depending on whether or not you collected both coins, you will either go down or go up 1 level.
This will change the modifiers for speed and jump.

After reaching either level 5 or level -5, you will win the game.
